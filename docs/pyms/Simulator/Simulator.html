pyms.Simulator.Function.add_gaussv_noise(im, scale, cutoff, prop)[source]¶
Summary

adds noise to an IntensityMatrix object

Parameters

  • im – the intensity matrix object

  • im – pyms.GCMS.Class.IntensityMatrix

  • scale (intType) – the scale of the normal distribution from which the noise is drawn

  • cutoff (intType) – The level below which the intensity of the ic at that point has no effect on the scale of the noise distribution

  • scale – The scale of the normal distribution for ic values

  • prop (floatType) – For intensity values above the cutoff, the scale is multiplied by the ic value multiplied by prop

Author

Sean O’Callaghan

pyms.Simulator.Function.add_gaussv_noise_ic(ic, scale, cutoff, prop)[source]¶
Summary

adds noise to an ic. The noise value is drawn from a normal distribution, the scale of this distribution depends on the value of the ic at the point where the noise is being added

Parameters

  • ic (pyms.GCMS.IonChromatogram) – The IonChromatogram

  • cutoff (intType) – The level below which the intensity of the ic at that point has no effect on the scale of the noise distribution

  • scale (intType) – The scale of the normal distribution for ic values below the cutoff is modified for values above the cutoff

  • prop (floatType) – For ic values above the cutoff, the scale is multiplied by the ic value multiplied by prop

Author

Sean O’Callaghan
