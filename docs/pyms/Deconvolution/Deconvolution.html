pyms.Deconvolution.BillerBiemann.Function.BillerBiemann(im, points=3, scans=1)[source]¶
Summary

BillerBiemann Deconvolution.

Deconvolution based on the algorithm of Biller and Biemann (1974)

Parameters

  • im (pyms.GCMS.Class.IntensityMatrix) – An IntensityMatrix object

  • points (IntType) – Peak if maxima over ‘points’ number of scans (Default 3)

  • scans (IntType) – To compensate for spectra skewing, peaks from ‘scans’ scans are combined (Default 1).

Returns

List of Peak objects

Return type

ListType
