pyms.Gapfill.Function.missing_peak_finder(sample, filename, points=13, null_ions=[73, 147], crop_ions=[50, 540], threshold=1000, rt_window=1, filetype='mzml')[source]¶
Summary

Integrates raw data around missing peak locations to fill in NAs in the data matrix

Parameters

  • sample (pyms.MissingPeak.Class.Sample) – The sample object containing missing peaks

  • andi_file (stringType) – Name of the raw data file

  • points (intType) – Peak finding - Peak if maxima over ‘points’ number of scans (Default 3)

  • null_ions (listType) – Ions to be deleted in the matrix

  • crop_ions (listType) – Range of Ions to be considered

  • threshold (intType) – Minimum intensity of IonChromatogram allowable to fill missing peak

  • rt_window (floatType) – Window in seconds around average RT to look for missing peak

  • filetype (stringType) – either mzml or netcdf

Author

Sean O’Callaghan
