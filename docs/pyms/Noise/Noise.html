pyms.Noise.Analysis¶

Noise analysis functions

pyms.Noise.Analysis.window_analyzer(ic, window=256, n_windows=1024, rand_seed=None)[source]¶
Summary

A simple estimator of the signal noise based on randomly placed windows and median absolute deviation

The noise value is estimated by repeatedly and picking random windows (of a specified width) and calculating median absolute deviation (MAD). The noise estimate is given by the minimum MAD.

Parameters

  • ic (pyms.IO.Class.IonCromatogram) – An IonChromatogram object

  • window (IntType or StringType) – Window width selection

  • n_windows (IntType) – The number of windows to calculate

  • rand_seed (IntType) – Random seed generator

Returns

The noise estimate

Return type

FloatType

Author

Vladimir Likic

Savitzky-Golay noise filter

pyms.Noise.SavitzkyGolay.__calc_coeff(num_points, pol_degree, diff_order=0)[source]¶
Summary

Calculates filter coefficients for symmetric savitzky-golay filter

See: http://www.nrbook.com/a/bookcpdf/c14-8.pdf

Parameters

  • num_points (IntType) – Means that 2*num_points+1 values contribute to the smoother

  • pol_degree (IntType) – The degree of fitting polynomial

  • diff_order – The degree of implicit differentiation. 0 means that filter results in smoothing of function, 1 means that filter results in smoothing the first derivative of function, and so on. Always use 0

Returns

Filter coefficients

Return type

numpy.ndarray

Author

Uwe Schmitt
